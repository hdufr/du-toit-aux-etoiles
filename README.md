# Du toit aux étoiles

## Description
Application web de lecture audio pour le projet "Du toit aux étoiles".

## Fonctionnalités
- Lecteur audio responsive
- Playlist dynamique
- Design épuré et élégant

## Technologies
- HTML5
- CSS3
- JavaScript
- Déployé sur Vercel

## Installation
1. Cloner le dépôt
2. Ouvrir `index.html` dans un navigateur
